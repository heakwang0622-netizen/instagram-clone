# 멍스타그램 프론트엔드 디자인 명세서

## 1) 디자인 목표
- 기존 Instagram/Kwangstagram 톤을 제거하고, **강아지 사진 공유 특화 서비스**인 `멍스타그램` 아이덴티티로 전환한다.
- 백엔드 API, 데이터 구조, 라우팅, 기능 플로우는 그대로 유지한다.
- 변경 범위는 프론트엔드의 브랜드/시각/문구/UI 아이콘으로 한정한다.

## 2) 색상 팔레트 (강아지 테마: 흰색/검은색 기반)
- `--ig-bg`: `#f7f7f7` (배경)
- `--ig-surface`: `#ffffff` (카드/패널)
- `--ig-border`: `#d6d6d6` (경계선)
- `--ig-text`: `#111111` (기본 텍스트)
- `--ig-text-secondary`: `#5e5e5e` (보조 텍스트)
- `--ig-link`: `#222222` (링크/강조)
- `--ig-link-hover`: `#000000` (링크 hover)
- `--ig-like`: `#111111` (반응 강조색: 멍멍)

### 톤 앤 무드
- 화려한 그라디언트 대신, 모노톤(화이트/블랙/그레이)으로 안정적이고 깔끔한 인상.
- 사진 콘텐츠 대비를 높이기 위해 배경은 밝게, 텍스트는 고대비 유지.

## 3) 브랜드 요소 변경

### 로고
- 기존 워드마크/그라디언트 로고 제거.
- `MeongstagramLogo` 컴포넌트로 교체:
  - `word` 모드: `멍스타그램` 텍스트
  - `glyph` 모드: 발바닥 형태 아이콘

### 파비콘
- 그라디언트 사각형 아이콘 제거.
- 흰 배경 + 검정 발바닥 아이콘으로 교체.

### UI 아이콘
- `NavIcons.tsx`의 주요 아이콘을 강아지 무드로 리디자인:
  - 홈: 강아지 집 형태
  - 탐색: 발바닥 형태
  - 릴스/미디어: 반려동물 카메라 느낌
  - 좋아요 대체 반응 아이콘 톤 다운 (모노톤)
- 모바일 헤더 메시지 아이콘도 공통 아이콘(`SendIcon`)으로 통일해 테마 일관성 확보.

## 4) 텍스트/용어 정책

### 서비스명
- `Instagram`, `Kwangstagram`, `Kwangsta`, `인스타그램` 표기를 모두 `멍스타그램`으로 통일.

### 주요 용어
- `팔로우` -> `구독`
- `팔로잉` -> `구독중`
- `좋아요` -> `멍멍`

### 반영 위치
- 피드/상세/탐색/릴스/알림/프로필/설정/관리자 페이지 내 사용자 가시 텍스트.
- 브라우저 탭 타이틀, 404 안내 문구, 목업 프로필 소개 문구.

## 5) 컴포넌트별 변경 사항

### 레이아웃
- `AppTopNav`, `Sidebar`, `MobileHeader`의 브랜드 컴포넌트를 `MeongstagramLogo`로 교체.
- 상단/사이드 내비게이션은 동일한 정보 구조를 유지하되, 시각 아이덴티티만 변경.

### 인증 화면
- 가입 안내 문구를 반려견 콘텐츠 중심으로 수정.
- 연락처 관련 안내 문구의 서비스명을 `멍스타그램`으로 통일.

### 피드/콘텐츠 카드
- 반응 지표 텍스트 `좋아요` -> `멍멍`.
- 추천/구독 동선에서 `팔로우` 계열 문구 -> `구독` 계열 문구.

### 프로필/관계 화면
- 프로필 행동 버튼: `팔로우/팔로잉` -> `구독/구독중`.
- 팔로우 목록 액션 버튼: `구독`으로 변경.

### 알림
- like/follow 타입 문구를 각각 `멍멍`, `구독` 중심 문구로 변경.

### 관리자 화면
- 통계/목록에서 `좋아요` 용어를 `멍멍`으로 통일 (데이터 의미는 동일).

## 6) 레이아웃 개선안 (기능 추가 없이 적용 가능한 범위)
- 카드 간 여백을 조금 넓혀 강아지 사진 중심의 시각 호흡 확보.
- 모서리 라운드와 경계선 대비를 살짝 강화해 카드 단위 탐색성 향상.
- 반응/액션 버튼 텍스트 대비를 높여 모바일 터치 타깃 인지성 향상.
- 피드 우측 보조영역(추천/안내)은 문구 길이를 줄여 핵심 CTA 집중.
